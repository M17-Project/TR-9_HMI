# TR-9_KBD
Firmware for the TR-9 keyboard microcontroller

UART: 9600baud 8n1  
  
Keys:  
- rotary encoder: 'L' for counter-clockwise, 'R' for clockwise rotation  
- 'A' for sub-display button left, 'B' for right  
- rows from bottom to the top: {'*', '0', '#'}, {'7', '8', '9'}, {'4', '5', '6'}, {'1', '2', '3'}
